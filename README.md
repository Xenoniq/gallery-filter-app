# PixelNest — премиальная галерея изображений с умной фильтрацией

Готовый итоговый проект по дисциплине **«Веб-дизайн»** на чистых **HTML, CSS и JavaScript**.

Проект не требует сборщика, стабильно работает на обычном хостинге и подходит для публикации на **GitHub Pages**.

## Что реализовано

### Обязательная часть задания

- фильтрация по **категории**;
- фильтрация по **тегам**;
- сохранение текущего состояния фильтров в **URL**;
- корректное восстановление интерфейса по ссылке;
- кнопка **«Поделиться ссылкой с фильтрами»**;
- fallback-логика шаринга: `Web Share API` → `Clipboard API` → ручное копирование;
- реальный контент: карточки работают не на заглушках, а на **настоящих изображениях**.

### Что добавлено сверх задания

- поиск по названию, описанию, категории и тегам;
- сортировка карточек;
- динамические счётчики рядом с категориями и тегами;
- живой hero-preview первой карточки из текущей выборки;
- полноэкранный просмотр изображений в lightbox;
- клавиатурная навигация в модальном окне (`Esc`, `←`, `→`);
- быстрый фокус на поиск по клавише `/`;
- адаптивная вёрстка для десктопа, планшета и мобильных устройств;
- резервное изображение на случай ошибки загрузки;
- чистая модульная архитектура без фиктивных элементов.

## URL-параметры

Используются query-параметры:

- `category` — выбранная категория;
- `tags` — теги через запятую;
- `sort` — текущий режим сортировки;
- `q` — строка поиска.

### Примеры

```text
./?category=travel&tags=sea,sunset
./?category=art&tags=museum,texture&sort=titleAsc
./?q=неон&sort=category
```
